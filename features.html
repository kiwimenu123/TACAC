Powerful Features

Everything you need to keep your FiveM server secure and cheat-free.

God Mode Detection

Advanced health monitoring system that tracks damage events and detects when players don't take proper damage. Catches invincibility exploits instantly.

Speed Hack Detection

Real-time velocity monitoring with configurable speed limits for on-foot, vehicle, and swimming. Tolerance settings to prevent false positives.

NoClip Detection

Detects players flying, floating, or phasing through walls. Uses advanced position and velocity checks to catch noclip exploits.

Teleport Detection

Monitors player positions and detects unauthorized instant teleportation. Configurable distance thresholds and spawn protection.

Weapon Blacklist

Block and remove blacklisted weapons automatically. Configurable weapon list including RPGs, miniguns, and other OP weapons.

Vehicle Blacklist

Prevent spawning of blacklisted vehicles like jets, tanks, and military aircraft. Automatically removes players from and deletes blacklisted vehicles.

Explosion Monitoring

Track all explosion events and detect excessive explosions. Configurable limits and banned explosion types.

Resource Injection

Detect when new resources are injected into the client. Protects against executor scripts and menu injections.

Infinite Ammo Detection

Monitors ammo counts and detects suspicious increases. Catches infinite ammo cheats automatically.

Super Jump Detection

Monitors jump heights and detects abnormal jumps. Prevents super jump exploits.

Ban System

Comprehensive ban system with temporary and permanent bans. Unique ban IDs, automatic expiry, and ban appeals integration.

Admin System

Multi-tier admin system with Moderator, Admin, and Super Admin roles. Admins automatically bypass detections.

Whitelist System

Flexible whitelist system to bypass specific or all detections. Perfect for trusted players and developers.

Discord Integration

Real-time Discord webhooks for detections, bans, kicks, and server status. Beautiful embeds with all relevant information.

Web Dashboard

Full-featured web dashboard to manage bans, admins, whitelist, view detections, and configure settings remotely.

Easy Configuration

Simple Lua configuration with detailed comments. Dashboard generates config automatically with your credentials.
